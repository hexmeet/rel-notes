会捷通 macOS 版 Release Notes

hexmeet

# 2.0.34 (发布日期 2024/06/02)

  • 会议中参会者关闭摄像头的显示方式优化
  • 支持客户端单设备登录
  • 会议分享信息支持携带直播地址和密码
  • 支持登出后或自动登录失败时自动清除密码
  • 支持以镜像方式显示本地视频
  • 修复服务器不可达时登录持续加载的问题
  • 修复设置1x2（左本地，右远端）布局后本地视频可能出现黑屏的问题
  • 修复设置内容+嵌入画中画视图，后续传统主讲视图丢失窗口的问题
  • 修复若干已知问题

# 1.8.3083 (发布日期 2024/01/13)

  • 修复OS 11 及以下系统版本上入会可能遇到异常退出的问题

# 1.8.3082 (发布日期 2024/01/05)

  • 修复应用新安装无登录信息时链接入会失败的问题
  • 修复在某布局下接收内容，当内容源发生变更后画中画黑屏的问题
  • 修复在登出状态多次打开摄像头预览画面卡顿的问题
  • 修复用户登录后获取不到部门、职务信息的问题
  • 修复普通用户跳转我的自服务概率报错的问题

# 1.8.3058 (发布日期 2023/12/19)

  • 优化新消息气泡提示的格式
  • 优化消息内容支持复制
  • 接收内容共享时切换嵌入式的新型主讲，优化翻页按钮的位置
  • 修复接收内容、抢发内容或被抢发内容后改变布局，本地视频或画中画可能会出错的问题
    • 修复接收内容共享时切换1x2布局无本地视频窗口的问题
    • 修复接收内容共享时切换1x2布局，内容源发生变更后本地视频窗口自动隐藏的问题
    • 修复接收内容共享时切换1x2布局，“更多”中有多余的“显示画中画”按钮的问题
    • 修复仅两方参会接收内容共享，从新型主讲切换至1x2时画中画显示黑屏的问题
    • 修复抢发内容共享，共享控制栏“显示画中画”按钮错误的问题
    • 修复被抢发双流后画中画远端视频为黑屏的问题
  • 修复应用启动后首次入会显示远端麦克风状态错误的问题
  • 修复打开消息窗口后，未读消息提示图标未及时消失的问题
  • 修复若干其他问题

# 1.8.3045 (发布日期 2023/12/06)

  • 视频窗口限定最小尺寸，并可拖拽/拉伸大小
  • 共享工具栏增加消息、分组讨论、参与投票、会议信息、统计信息等的入口
  • 支持视频分屏布局：1x2（本地和演讲者均衡）、1P5、1TP5B、1TP6B、1P7
  • 支持接收双流时设置1x2（内容和演讲者均衡）布局
  • 支持用户名带空格时自动截取
  • 视图布局选择优化，布局样式悬浮窗弹出一键选择
  • 会议中水印优化，对比度提高，在浅色视频或内容接收时更明显
  • 优化工具栏上或顶部状态栏当鼠标在其上时常驻，鼠标移开后自动隐藏
  • 优化网络状况不佳的提示，支持网络重连提示
  • 修复若干已知问题

# 1.8.2094 (发布日期 2023/09/04)

  • 登录界面改进，统一云和私有部署入口
  • 登录界面增加“保持登录”开关
  • 支持会场站点名称样式由会议统一管控
  • 支持会议延时提醒，增加延时操作入口
  • 支持消息敏感词过滤（匹配平台v2.0.0）
  • 会议录制控制优化，工具栏增加录制入口
  • 优化呼叫历史展示会议名称
  • 会议锁定状态展示优化
  • 入会过程界面过渡体验优化
  • 增加被叫振铃时的动态效果
  • 修复若干已知问题

# 1.8.131 (发布日期 2023/06/28)

  • 支持网络研讨会
  • 优化弱网条件下的用户体验
  • 优化签到窗口显示签到标题
  • 优化接收内容共享时视频小窗口的分辨率上限，不挤占主窗口带宽
  • 优化外呼时去除振铃
  • 优化会中复制会议信息空白项不予展示
  • 优化解除静音确认框的文字和按钮
  • 优化无麦克风设备时给与文字提醒
  • 优化网络检测服务器无响应时的错误提示
  • 修复共享全屏状态的应用，远端收到黑屏的问题
  • 修复反复点击麦克风按钮可能会弹出解除静音确认窗口的问题
  • 修复网络检测时固定使用首个线路的问题
  • 修复若干其他问题

# 1.7.26 (发布日期 2023/2/1)

  • 支持虚拟背景和背景虚化（macOS 12.0及以上）
  • 支持消息窗口合并到主窗口
  • 支持视频硬解码
  • 优化会中消息的接收、发送
  • 优化会中改名后，消息名称及时变更
  • 修复发起随机号码的会议按钮不工作的问题
  • 修复跨可用区设置消息权限不生效的问题
  • 修复共享内容时画中画不支持最小化的问题
  • 修复“参会者姓名大小”设置项某些情况下未起效的问题
  • 修复有未读消息“更多”按钮未标记红点的问题
  • 修复若干其他问题

# 1.6.2395 (发布日期 2022/9/24)

  • UI和UX的整体改进
  • 接收内容分享时，支持显示发送方名称的功能
  • 优化用户及入会名称的长度上限，统一32个字符以内
  • 优化静止长字幕的展示方式，无法展示完全时做到左对齐
  • 优化以视频窗口宽度来适应展示用户的名称长度
  • 修复平台侧修改客户端配置后，App重新登录后未生效问题
  • 修复发起共享时视频窗口仍可调出的问题
  • 修复在等候室点击窗口最顶端会弹出空内容的会议信息卡片
  • 修复若干其他问题

# 1.6.456 (发布日期 2022/7/15)

  • 优化发起屏幕共享时检查屏幕录制权限的机制
  • 优化音频16kHz采样检测
  • 优化唇音同步
  • 优化网络乱序计算问题
  • 修复手机号-验证码登录后未获得功能项的问题
  • 修正媒体统计信息中的竖屏视频分辨率
  • 修复抢发屏幕共享后，之前的内容接收窗口未及时关闭的问题
  • 修复若干其它问题

# 1.6.420 (发布日期 2022/5/21)

  • 新增对服务端侧如下功能开关的支持：允许普通用户登录自服务、会议分享权限、初次登录修改密码、启动双因子登录
  • 新增对服务端推送会议变更通知自动更新会议列表的功能
  • 共享控制栏麦克风、摄像头设置窗口的关闭优化
  • 修复共享控制栏麦克风状态与本地麦克风状态未同步的问题
  • 通讯录展现或隐藏做到对重新登录起效
  • 平台公告在App侧展示做到对每次登录起效
  • 重新登录时自动刷新会议列表
  • 修复私密消息在某些特殊情况下出现重复的问题
  • 修复会中私密消息发送在某些情况下未自动重置回公开消息的问题
  • 修复通过平台中心区呼叫不存在的会议时文字提示不明确的问题
  • 修复无网络检测服务器时，打开“设置”报“无效服务器”的问题
  • 修复登录后点击“设置”摄像头指示灯自动点亮的问题
  • 修复更新进程被取消后，再次触发下载新版本报失败的问题
  • 修复p2p主叫取消呼叫，被叫未自动结束振铃的问题
  • 特殊场景的处理优化：平台升级过程中或服务端可用区停用后App收到相应消息自动退出登录状态、服务端可用区停用后App尝试登录的提示优化
  • 修复无空闲会议室导致发起快速会议失败时App未明示呼叫失败的问题
  • 修复若干其它问题

# 1.6.380 (发布日期 2022/4/16)

  • 新增入会麦克风/摄像头默认状态的通用设置
  • 新增共享应用，及共享窗口的截屏功能

  • 修复无立即召开会议权限时，尝试发起立即会议导致App崩溃的问题
  • 修复主会场模式终端侧展示视频页码的问题
  • 修复1080p设置未生效问题
  • 修复若干其它问题
