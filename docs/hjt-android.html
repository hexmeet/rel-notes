已知问题

通过 Chrome 浏览器 （版本 72.0） 打开会议邀请页面，点击加入会议时，有可能无法打开 App 入会。建议升级 Chrome 浏览器或者尝试改用其他浏览器打开会议邀请页面。

# 1.2.54 (发布日期 2019/03/28)

  • 音视频内核升级，进一步改进音视频通话质量
  • 进一步适配 安卓 9.0 系统
  • 改进 APP 前后台切换场景的处理，缓解后台运行状态容易被系统回收的现象
  • 当会议被直播时，显示直播状态
  • 解决有时声音可能转到听筒的问题
  • 解决使用蓝牙耳机的一些问题
