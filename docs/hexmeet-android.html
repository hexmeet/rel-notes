HexMeet Android 版 Release Notes

hexmeet

# 2.8.419 (发布日期 2025/01/03)

  • 支持浮窗告知应用申请相机、麦克风、通话状态权限的目的
  • 优化“退出登录”按钮移至个人信息页
  • 修复上传头像时报错的问题
  • 优化呼叫记录，修复记录列表可能出现展示错误的情况

# 2.8.281 (发布日期 2023/04/17)

  • 优化“HexMeet隐私政策”提示弹窗
  • 优化麦克风已静音提示间隔10秒
  • 修复过度申请"CALL_PHONE"权限的问题
  • 修复意见反馈会重复申请存储权限的问题

# 2.8.225 (发布日期 2022/10/09)

  • 适用小米市场的隐私政策及权限调整

# 2.8.212 (发布日期 2022/08/23)

  • 修复录音权限问题(初次安装入会可能遇到unmute后发言不能被远端听到)

# 2.8.191 (发布日期 2022/07/05)

  • 修复低版本Android系统 （如Android 6.0）上的闪退问题

# 2.8.182 (发布日期 2022/06/20)

  • 悬浮窗不再要求开启权限
  • P2P音视频呼叫去掉“与会者”按钮
  • 修复Android 12系统上偶发的闪退问题

# 2.8.171 (发布日期 2022/05/26)

  • 修复登录服务器输入特殊符号等错误地址，点击登录会闪退的问题
  • 修复Android 12系统上开启硬解码入会后远端视频黑屏的问题

# 2.8.167 (发布日期 2022/05/24)

  • 修复全新安装点击"设置"->"关于"->"版本更新"会闪退的问题

# 2.8.161 (发布日期 2022/05/19)

  • 兼容Android 12手机系统
  • 修复视频通话中后置摄像头角度不正确的问题
  • 去掉了登录CCM后通讯录和呼叫历史页面展示的默认头像

# 2.8.144 (发布日期 2022/04/24)

  • 修复特定手机上视频通话中接听电话引起静音状态不对的问题
  • 全新UX设计，新增呼叫历史等功能

# 2.6.417 (发布日期 2022/03/01)

  • 修复端口号输入非数字引发登录crash的问题

# 2.6.416 (发布日期 2022/02/28)

  • 修复弱网情况下会议中静音状态偶尔显示错误的问题

# 2.6.415 (发布日期 2022/02/15)

  • 修复特定情况下的登录crash
  • 修改App首次安装的权限设置，禁止请求权限后允许登录App

# 2.6.345 (发布日期 2021/09/09)

  • 修复接听SIM卡电话后，返回视频通话页面会自动打开本地静音的问题

# 2.6.292 (发布日期 2021/03/30)

  • 更新闪屏图
  • 修复不显示最小化视频窗口按钮的问题

# 2.6.271 (发布日期 2021/03/30)

  • 登录界面改进
  • 解决若干其他问题

# 2.6.196 (发布日期 2021/01/01)

  • 解决下载升级更新版本时可能卡住的问题

# 2.6.192 (发布日期 2020/12/26)

  • 支持更改密码时校验密码复杂度规则功能 （配合 CCM 平台）
  • 解决若干其他问题

# 2.6.177 (发布日期 2020/11/27)

  • 修复客户端静音状态可能和实际不一致的问题
  • 修复进行微信语音后，会议声音可能被错误的改为听筒的问题
  • 修复接听手机电话后，麦克风和扬声器不起作用的问题
  • 修复若干其他问题

# 2.6.158 (发布日期 2020/09/29)

  • 解决 4G 网络时，设置呼叫速率导致崩溃的问题
  • 解决自动登录时，有机会点击登录按钮，并导致无法注册等异常情况的问题
  • 解决呼叫速率设置可能不生效的问题

# 2.6.143 (发布日期 2020/09/15)

  • 解决若干与 CCM 配合时的问题

# 2.6.136 (发布日期 2020/09/05)

  • 解决某些场景不能呼叫带密码会议的问题

# 2.6.126 (发布日期 2020/08/27)

  • 解决若干问题

# 2.6.120 (发布日期 2020/08/16)

  • 解决 2.6.106 存在的几个有关 2.5 升级 2.6 后出现的登录问题

# 2.6.106 (发布日期 2020/08/15)

  • SDK 更新为 EVSDK
  • UI 实现内部优化
  • 解决若干其他问题

# 2.5.76 (发布日期 2020/04/19)

  • 解决与 COP 模式 MCU 互通分辨率帧率不足问题

# 2.5.75 (发布日期 2020/04/11)

  • 解决进入 RMX 会议失败的问题

# 2.5.74 (发布日期 2020/04/08)

  • 解决入会后会议管理按钮可能没有显示的问题 （CCM 场景）

  • 解决入会后会议管理页面内容可能显示异常的问题 （CCM 场景）

  • 其他优化

# 2.5.63 (发布日期 2020/02/15)

  • 解决使用非 443 端口登录 CCM 时，无法加载会议页面的问题
  • 修复若干其它问题

# 2.5.38 (发布日期 2019/09/18)

  • 音视频质量改进
  • 支持登录 CCM
  • 音频支持新的算法，改善丢包时的声音质量
  • 修复若干其它问题

# 2.5.14 (发布日期 2019/06/14)

  • 音视频质量改进
  • 进一步适配 安卓 9.0 系统
  • 修复若干其它问题
