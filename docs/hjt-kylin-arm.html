会捷通 Kylin Arm64 版 Release Notes

hexmeet

# 2.1.0.66 (发布日期 2025/03/28)

  • 支持共享应用窗口
  • 支持暂停/恢复共享
  • 改进降噪和啸叫抑制算法，降低CPU消耗
  • 优化下载远端头像超时时长为18秒，最多重试10次
  • 修复入会时视频有时脱出窗口的问题
  • 修复上报MAC地址有误或未上报的问题
  • 修复若干其他问题

# 2.0.0.23 (发布日期 2024/10/28)

  • 登录界面改进，统一云和私有部署入口
  • 登录界面增加“保持登录”开关
  • 支持用户名带空格自动截取
  • 支持客户端单设备登录
  • 支持登出后或自动登录失败时自动清除密码
  • 支持会场站点名称样式由会议统一管控
  • 支持会议延时提醒，增加延时操作入口
  • 支持消息敏感词过滤
  • 支持接收双流时设置1x2布局，且双击远端视频支持选定
  • 会议分享信息支持携带直播地址和密码
  • 支持以镜像方式显示本地视频
  • 入会过程界面过渡体验优化
  • 会议中参会者关闭摄像头的显示方式优化
  • 会议录制控制优化，工具栏增加录制入口
  • 优化呼叫历史展示会议名称
  • 会议锁定状态展示优化

# 1.8.0.9 (发布日期 2023/04/23)

  • UI和UX的整体改进
  • 支持密信SSO登录
  • 支持网络研讨会
  • 选定视频优化，所有布局下均可双击选定视频
  • 优化网络检测时应用的稳定性
  • 优化会中消息的接收、发送，及消息名称的及时变更
  • 优化签到窗口显示签到标题
  • 优化字幕字数较多时的展示方式
  • 修复关闭应用时耗时过长的问题
  • 修复长时间入会后视频会定格的问题
  • 修复不能识别第二个相同摄像头的问题
  • 修复加入新会或进入分组会议后录制/直播图标始终展示的问题
  • 修复若干其他已知问题
