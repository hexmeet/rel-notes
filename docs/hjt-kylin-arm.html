# 1.8.0.9 (发布日期 2023/04/23)

  • UI和UX的整体改进
  • 支持密信SSO登录
  • 支持网络研讨会
  • 选定视频优化，所有布局下均可双击选定视频
  • 优化网络检测时应用的稳定性
  • 优化会中消息的接收、发送，及消息名称的及时变更
  • 优化签到窗口显示签到标题
  • 优化字幕字数较多时的展示方式
  • 修复关闭应用时耗时过长的问题
  • 修复长时间入会后视频会定格的问题
  • 修复不能识别第二个相同摄像头的问题
  • 修复加入新会或进入分组会议后录制/直播图标始终展示的问题
  • 修复若干其他已知问题
