会捷通 iPhone/iPad 版 Release Notes

hexmeet

# 2.1.47 (发布日期 2025/01/24)

  • 支持啸叫抑制
  • 支持智能降噪，设置低、中、高、增强四个降噪等级
  • 修复网络检测会导致崩溃的回退问题
  • 修复全新安装应用，首次发起会议会崩溃的问题
  • 修复会中转移时可能出现的崩溃问题
  • 会中消息恢复复制功能、展示收发时间点

# 2.1.18 (发布日期 2024/12/05)

  • 会中消息体验改进（使用Flutter重构）
  • 修复在iOS 18 (2024/09/17 发布) 上入会崩溃的问题
  • 修复观众身份入会App会崩溃的回退问题
  • 修复若干其它问题

# 2.0.91 (发布日期 2024/09/10)

  • 支持以镜像方式显示本地视频
  • 支持TLS1.3
  • 优化切换摄像头按钮的展示场景
  • 修复会议链接调起应用不能完成自动登录的问题
  • 修复被抢双流后，退会时崩溃的问题
  • 修复在主讲视图下切换布局可能无响应的问题
  • 修复若干已知问题

# 2.0.51 (发布日期 2024/06/13)

  • 会议中参会者关闭摄像头的显示方式优化
  • 支持客户端单设备登录
  • 会议分享信息支持携带直播地址和密码
  • 优化检查更新直接跳转App Store
  • 修复旋转横竖屏后本地视频可能不可见的问题
  • 修复嘉宾链接入会“会议嘉宾”登录弹窗不弹出的问题
  • 修复若干已知问题

# 1.8.2254 (发布日期 2024/04/22)

  • 支持虚拟背景和背景虚化（iOS 15以上）
  • 支持用户名带空格时自动截取
  • 优化未读消息条数及时刷新
  • 修复在低版本iOS（如 12.x）设备启动崩溃的问题

# 1.8.2192 (发布日期 2024/01/03)

  • 支持麦克风悬浮按钮展示或隐藏的设置开关
  • 支持会中本地窗口拖动位置
  • 优化语音模式下工具栏固定展示
  • 优化使用邮箱作为账号登录后可自动登录
  • 优化入会时展示会控布局
  • 修复进消息窗口时闪现入会前的界面的问题
  • 修复应用安装后首次入会本地摄像头和远端麦克风状态错误的问题
  • 修复若干已知问题

# 1.8.2136 (发布日期 2023/11/08)

  • 增加常驻的麦克风图标按钮
  • 会议中水印优化，对比度提高，在浅色视频或内容接收时更明显
  • 优化网络状况不佳的提示，支持网络重连提示
  • 优化签到窗口显示签到标题
  • 优化语音模式工具栏常驻
  • 修复若干已知问题

# 1.8.2093 (发布日期 2023/09/16)

  • 登录界面改进，统一云和私有部署入口
  • 支持会场站点名称样式由会议统一管控
  • 支持会议延时提醒，增加延时操作入口
  • 支持消息敏感词过滤（匹配平台v2.0.0）
  • 优化呼叫历史展示会议名称
  • 优化网络状况不佳的提示，支持网络重连提示
  • 会议锁定状态展示优化
  • 入会过程界面过渡体验优化
  • 修复若干其他问题

# 1.8.111 (发布日期 2023/06/30)

  • 支持网络研讨会
  • 支持消息横屏输入
  • 优化弱网条件下的用户体验
  • 优化解除静音确认框的文字和按钮
  • 优化视频翻页时本地视频窗口不随动
  • 优化视频翻页过渡色为黑色
  • 修复退会后底部导航栏可能消失的问题
  • 修复反复点击麦克风按钮可能会弹出解除静音确认窗口的问题
  • 修复若干其他问题

# 1.7.21 (发布日期 2023/3/28)

# 1.7.19 (发布日期 2023/3/24)

  • 修复呼叫频繁错误码翻译丢失的问题
  • 修复会控更改滚动字幕高度时App侧字幕消失的问题

# 1.7.15 (发布日期 2023/3/21)

  • 优化退会后号码输入框不再展示会议号码
  • 修复企业微信唤起入会的问题
  • 修复在会中非首页时增加与会者会导致视频页码出错的问题
  • 修复会议模式切换、音频视频切换、主讲画廊切换后画廊未自动恢复2x2的问题
  • 修复发送弹幕消息时会多显示一条无关消息的回退问题

# 1.7.8 (发布日期 2023/2/18)

  • UI和UX的整体改进
  • 优化会中消息的接收、发送
  • 优化会中改名后，消息名称及时变更
  • 优化用户及入会名称的长度上限，统一32个字符以内
  • 优化媒体统计信息中的带宽检测数据展示位置
  • 支持视频硬解码
  • 修复“申请发言”按钮始终展示的问题
  • 修复从消息界面退出后视频被锁定竖屏的问题
  • 修复从分组返回主会时摄像头自动开启的问题
  • 修复退出会议后有时导航栏隐藏的问题
  • 修复接听新会时会议结束提醒窗口未自动关闭的问题
  • 修复若干其它问题

# 1.6.320 (发布日期 2022/12/21)

  • 修复在iOS系统升级后出现的邮件分享内容错误的问题
  • 修复同时连接两个蓝牙设备时入会崩溃
